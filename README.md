## クラス (P.117-123)

### そもそもクラスって何？
・「クラス」は独自の状態やメソッドを持つオブジェクトを生成するための「型」
・クラスから生成された個々のオブジェクトを「インスタンス」と呼びます

### 【具体例】
![](images/image1.png)
・「学生情報」というクラスと「ぬこさん」「とりさん」というインスタンスがある<br>
・「ぬこさん」「とりさん」に「学生情報」という型を使って情報を付与する行為 → ***インスタンス化***
